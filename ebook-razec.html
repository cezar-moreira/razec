Capítulo 01
🎯 O que é o RAZEC?
Conheça o sistema e entenda como ele pode transformar sua organização
💡
O RAZEC é o seu segundo cérebro digital — um lugar único para guardar projetos, ideias, notas e conhecimento, acessível de qualquer dispositivo.
O que você pode fazer
FuncionalidadeO que faz
🚀 Sem loginAbre e já funciona — nenhuma conta necessária
📁 ProjetosAgrupe notas por área; crie e exclua projetos com confirmação
📝 NotasEscreva em Markdown com preview; exclua pelo editor ou lista de arquivos
📑 Abas (tabs)Abra múltiplas notas ao mesmo tempo em abas separadas
🔍 Busca AvançadaFiltros por projeto:, tag:, tipo: e texto livre combinados
🕸️ Mapa D3.jsGrafo interativo estilo Obsidian — zoom, arraste nós, auto-fit mobile
📁 Drag & DropArraste arquivos .py/.js/.html/.sql/.md para importar
⌨️ 16 AtalhosCtrl+N, S, F, W, P, U, G, L, 1–5, Shift+P, / e Escape
📱 MobileBottom nav + botão FAB flutuante para celular e tablet
💾 Backup JSONExporte todos seus dados a qualquer momento
Como os dados ficam seguros?
🔒 Dados só no seu dispositivo
Tudo fica no localStorage do seu navegador. Nenhum servidor externo acessa seus dados.
🔐 HTTPS + GitHub Pages
Conexão criptografada via HTTPS. O app é servido pelo GitHub Pages com certificado SSL.

Capítulo 02
🚀 Como começar — sem login
Abra o app e comece a usar imediatamente — sem criar conta
🎉
O RAZEC não precisa de login nem cadastro. Abra o app e já pode usar — os dados ficam salvos no localStorage do seu navegador.
Primeiros passos
1

Acesse o app

Abra o navegador e acesse cezar-moreira.github.io/razec

2

Dashboard aparece imediatamente

Sem tela de login — você já está dentro do app

3

Crie um projeto

Clique em + Projeto no topo ou pressione Ctrl+Shift+P

4

Crie sua primeira nota

Pressione Ctrl+N ou clique no ＋ na barra de abas

Armazenamento local
💾
localStorage do navegador
Os dados ficam salvos no navegador deste dispositivo. Limite de 5MB — suficiente para milhares de notas. O indicador de uso aparece no rodapé da sidebar.
⚠️ Importante

Os dados ficam neste dispositivo. Para usar em outro, exporte o JSON e importe lá. Não limpe os dados do navegador sem fazer backup antes.

Salve

Capítulo 09
💾 Backup e Exportação de Dados
Seus dados estão seguros na nuvem — e você pode baixar uma cópia quando quiser
Onde ficam seus dados?
Todos os dados ficam no localStorage do seu navegador — sem servidor, sem nuvem obrigatória.
✅
Salvos automaticamente enquanto você digita
✅
Ficam salvos mesmo ao fechar o navegador
✅
Limite de 5MB — suficiente para milhares de notas
⚠️
Ficam neste dispositivo — use export para migrar
Exportar backup JSON
1

Encontre o botão de exportar

No menu ou configurações do app, procure "Exportar dados" ou "Backup JSON"

2

Clique em "Exportar JSON"

Um arquivo razec-backup.json é baixado automaticamente

3

Guarde o arquivo

Salve em local seguro (Google Drive, Dropbox, HD externo)

O que o arquivo JSON contém?
{
  "exportado_em": "2024-01-15T10:30:00Z",
  "projetos": [{ nome, ícone, cor, criado... }],
  "notas": [{ titulo, conteudo, tipo, tags, projeto... }]
}
💡 Recomendação

Faça um backup manual uma vez por mês. Se limpar os dados do navegador ou trocar de dispositivo sem exportar, os dados se perdem — o JSON é a única cópia de segurança.

Perguntas frequentes
Preciso criar conta?
Não. O app abre e funciona direto. Sem cadastro, sem login, sem senha.
Quantas notas posso criar?
O localStorage tem 5MB por navegador — suficiente para milhares de notas de texto. O indicador de uso aparece no rodapé da sidebar.
Meus dados se perdem se eu limpar o navegador?
Sim. Faça backup com Exportar JSON antes de limpar dados do navegador ou trocar de dispositivo.
O mapa ficou pequeno no celular, o que faço?
Clique em ↺ Resetar — o mapa faz auto-fit e se ajusta ao tamanho da tela automaticamente.
Funciona no iPhone/Android?
Sim. Chrome, Safari, Firefox, Edge. A interface tem bottom nav e botão FAB especialmente para mobile.

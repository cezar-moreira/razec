Capítulo 01
🎯 O que é o RAZEC?
Conheça o sistema e entenda como ele pode transformar sua organização
💡
O RAZEC é o seu segundo cérebro digital — um lugar único para guardar projetos, ideias, notas e conhecimento, acessível de qualquer dispositivo.
O que você pode fazer
FuncionalidadeO que faz
📁 ProjetosAgrupe suas notas por área (trabalho, estudos, saúde, etc.)
📝 NotasEscreva com suporte a Markdown — títulos, listas, negrito e mais
🔍 Busca AvançadaEncontre qualquer nota por texto, tag, projeto ou tipo
🕸️ Mapa VisualVeja como suas notas se conectam em um gráfico interativo
🤖 Gerador de PromptsCrie prompts prontos para usar no ChatGPT, Claude ou Gemini
☁️ NuvemDados salvos automaticamente — acesse de qualquer lugar
⚡ Sync Real-timeAbra no celular e no PC ao mesmo tempo — tudo sincroniza
💾 Backup JSONExporte todos seus dados a qualquer momento
Como os dados ficam seguros?
🔒 Isolamento por usuário
Cada conta só acessa os próprios dados. Nenhum usuário vê dados de outro.
🔐 HTTPS + Supabase
Conexão criptografada. Senhas armazenadas com hash bcrypt — nunca em texto.

🔑 Recuperação de Senha
Esqueceu a senha? Redefina em poucos cliques pelo email
Se você esquecer a senha, o RAZEC envia um link de redefinição para o seu email. Ao clicar no link, o app abre automaticamente com um formulário para você criar uma nova senha.
Passo a passo — redefinir senha
1

Clique em "Esqueci minha senha"

O link aparece abaixo do botão Entrar na tela de login

2

Digite seu email

Informe o email cadastrado na sua conta e clique em "Enviar link de recuperação"

3

Verifique seu email

Chegará um email com um link. Clique nele — ele abre o RAZEC automaticamente

4

Digite a nova senha

O app exibe o formulário de "Nova senha". Digite e confirme a nova senha

5

Pronto — já entrou!

Após salvar, você é logado automaticamente com a nova senha

Capítulo 09
💾 Backup e Exportação de Dados
Seus dados estão seguros na nuvem — e você pode baixar uma cópia quando quiser
Onde ficam seus dados?
Todos os dados ficam no Supabase — um banco de dados PostgreSQL na nuvem. Isso significa que:
✅
Não dependem do seu computador
✅
Ficam salvos mesmo se você fechar o navegador
✅
Acessíveis de qualquer dispositivo
✅
Auto-save a cada 2,5 segundos enquanto você digita
Exportar backup JSON
1

Encontre o botão de exportar

No menu ou configurações do app, procure "Exportar dados" ou "Backup JSON"

2

Clique em "Exportar JSON"

Um arquivo razec-backup.json é baixado automaticamente

3

Guarde o arquivo

Salve em local seguro (Google Drive, Dropbox, HD externo)

O que o arquivo JSON contém?
{
  "exportado_em": "2024-01-15T10:30:00Z",
  "projetos": [{ nome, ícone, cor, criado... }],
  "notas": [{ titulo, conteudo, tipo, tags, projeto... }]
}
💡 Recomendação

Faça um backup manual uma vez por mês. Os dados estão seguros na nuvem, mas ter uma cópia local dá tranquilidade adicional.

Perguntas frequentes
Esqueci minha senha, o que faço?
Na tela de login clique em "Esqueci minha senha", informe seu email e clique em "Enviar link". Chegará um email — clique no link e defina uma nova senha. Verifique o spam se não encontrar.
Posso usar sem internet?
O app precisa de conexão para carregar e salvar dados. Sem internet, os dados não são exibidos.
Quantas notas posso criar?
O plano gratuito do Supabase suporta 500MB — o suficiente para centenas de milhares de notas de texto.
Minhas notas são privadas?
Sim. O sistema usa Row Level Security — apenas você acessa seus dados com suas credenciais.
O app funciona no iPhone/Android?
Sim. Funciona em qualquer navegador moderno — Chrome, Safari, Firefox, Edge. Sem instalar nada.

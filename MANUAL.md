# Manual do RAZEC
### Guia completo de uso — do zero ao avançado

---

## O que é o RAZEC?

O RAZEC é sua central pessoal de projetos e conhecimento. Pense nele como um caderno digital inteligente que:

- Fica salvo na **nuvem** (não perde se trocar de computador)
- Funciona no **celular, tablet e computador**
- Organiza suas ideias em **projetos e notas**
- Conecta tudo em um **mapa visual**
- Gera **prompts prontos** para usar em IAs como ChatGPT e Claude

**Acesse em:** https://cezar-moreira.github.io/razec/

---

## Parte 1 — Primeiros Passos

### 1.1 Criando sua conta

1. Abra o app no navegador
2. Clique em **"Criar conta"**
3. Digite seu email e uma senha (mínimo 6 caracteres)
4. Clique no botão **"Criar conta"**
5. Pronto — você já entra automaticamente

> A mesma conta funciona em todos os seus dispositivos. Use o mesmo email e senha no celular, tablet ou outro computador.

### 1.2 Fazendo login

1. Abra o app
2. Certifique-se que a aba **"Entrar"** está selecionada
3. Digite email e senha
4. Clique em **"Entrar"**

### 1.3 A tela principal

Ao entrar, você vê três áreas:

## Parte 2 — Projetos

Projetos são **pastas** que organizam suas notas. Crie um projeto para cada área da sua vida ou trabalho.

### 2.1 Criar um projeto

1. No menu lateral, clique em **"Projetos"**
2. Clique no botão **"+ Novo Projeto"**
3. Preencha:
   - **Nome** — ex: "Marketing Digital"
   - **Descrição** — ex: "Estratégias e conteúdos para redes sociais"
   - **Ícone** — escolha um emoji (ex: 📊)
   - **Cor** — escolha uma cor para identificar visualmente
4. Clique em **"Salvar"**

### 2.2 Editar um projeto

1. Vá em **"Projetos"**
2. Clique no projeto que deseja editar
3. Clique no ícone de lápis (✏️)
4. Faça as alterações e salve

### 2.3 Excluir um projeto

1. Vá em **"Projetos"**
2. Clique no projeto
3. Clique no ícone de lixeira (🗑️)
4. Confirme a exclusão

> **Atenção:** Excluir um projeto não exclui as notas dentro dele. As notas ficam sem projeto associado.

## Parte 3 — Notas

Notas são onde você escreve seu conhecimento. Cada nota tem título, conteúdo, tipo, projeto e tags.

### 3.1 Criar uma nota

1. No menu lateral, clique em **"Notas"**
2. Clique em **"+ Nova Nota"**
3. Preencha:
   - **Título** — nome curto e descritivo
   - **Tipo** — nota, tarefa, referência, ideia ou resumo
   - **Projeto** — qual projeto essa nota pertence (opcional)
   - **Tags** — palavras-chave separadas por vírgula (ex: marketing, instagram, 2024)
   - **Fonte IA** — se o conteúdo foi gerado por IA, registre qual
   - **Conteúdo** — o texto da nota (suporta Markdown)
4. O conteúdo é **salvo automaticamente** enquanto você digita

### 3.2 Editar uma nota

1. Vá em **"Notas"**
2. Clique na nota que deseja editar
3. Faça as alterações — o auto-save salva em 2,5 segundos

### 3.3 Excluir uma nota

1. Abra a nota
2. Clique no ícone de lixeira (🗑️) no cabeçalho
3. Confirme a exclusão

### 3.4 Usando Markdown no conteúdo

O editor suporta Markdown. Exemplos:

| O que você digita | Como aparece |
|---|---|
| `# Título grande` | Título grande |
| `## Título médio` | Título médio |
| `**texto em negrito**` | **texto em negrito** |
| `*texto em itálico*` | *texto em itálico* |
| `- item de lista` | • item de lista |
| `1. item numerado` | 1. item numerado |
| `` `código` `` | `código` |
| `[link](https://...)` | link clicável |

## Parte 4 — Busca Avançada

A busca encontra notas por texto, projeto, tag ou tipo.

### 4.1 Busca simples

1. Clique em **"Busca"** no menu
2. Digite qualquer palavra — o sistema procura em títulos e conteúdos

### 4.2 Filtros especiais

Use prefixos para buscas precisas:

| Filtro | Exemplo | O que faz |
|---|---|---|
| `projeto:` | `projeto:instagram` | Mostra só notas do projeto Instagram |
| `tag:` | `tag:marketing` | Mostra notas com a tag "marketing" |
| `tipo:` | `tipo:tarefa` | Mostra só tarefas |
| texto livre | `reunião` | Busca em título e conteúdo |

### 4.3 Combinando filtros

Você pode combinar filtros com texto:
- `projeto:ultravida reunião` — notas do projeto Ultravida que mencionam "reunião"

## Parte 5 — Mapa de Conhecimento

O mapa mostra visualmente como suas notas e projetos se conectam.

### 5.1 Abrindo o mapa

1. Clique em **"Mapa"** no menu lateral
2. O mapa carrega automaticamente com todas as notas e projetos

### 5.2 Navegando no mapa

- **Clique em um nó** — abre a nota ou filtra pelo projeto
- **Passe o mouse** sobre um nó — mostra o título
- Os nós coloridos são **projetos**
- Os nós menores são **notas**
- Linhas conectam notas aos seus projetos

## Parte 6 — Gerador de Prompts

Use o Gerador de Prompts para criar instruções prontas para IAs como ChatGPT, Claude ou Gemini.

### 6.1 Gerando um prompt

1. Clique em **"Prompts"** no menu
2. Escolha o **tipo de prompt**:
   - Resumir nota
   - Expandir ideia
   - Criar plano de ação
   - Revisar texto
   - Gerar perguntas
3. Selecione a **nota de referência** (opcional)
4. Clique em **"Gerar Prompt"**
5. Copie o resultado e cole na IA de sua preferência

## Parte 7 — Dashboard

O Dashboard é a tela inicial — um resumo do que você tem no sistema.

### O que aparece no Dashboard

- **Total de projetos** ativos
- **Total de notas** criadas
- **Notas recentes** — as últimas notas modificadas
- **Distribuição por tipo** — quantas notas de cada tipo você tem

## Parte 8 — Dados e Backup

### 8.1 Onde ficam seus dados

Seus dados ficam no Supabase, um banco de dados PostgreSQL na nuvem. Eles **não ficam** no seu computador — por isso funcionam em qualquer dispositivo.

### 8.2 Exportar backup

1. Clique no ícone de configurações (⚙️) ou no menu de dados
2. Clique em **"Exportar JSON"**
3. Um arquivo `.json` é baixado com todas as suas notas e projetos

### 8.3 Importar dados

1. Clique em **"Importar JSON"**
2. Selecione o arquivo `.json` exportado anteriormente
3. Os dados são mesclados com os existentes

## Parte 9 — Acessando em outros dispositivos

### No celular ou tablet

1. Abra o navegador (Chrome, Safari, etc.)
2. Acesse: `https://cezar-moreira.github.io/razec/`
3. Faça login com o mesmo email e senha
4. Todos os seus dados aparecem instantaneamente

### Sincronização em tempo real

Se você abrir o app em dois dispositivos ao mesmo tempo e criar uma nota em um deles, o outro atualiza automaticamente em segundos — sem precisar recarregar a página.

## Parte 10 — Segurança e Privacidade

- Sua senha nunca é armazenada em texto — o sistema usa criptografia bcrypt
- Cada usuário só acessa seus próprios dados (Row Level Security)
- Conexão sempre via HTTPS (criptografada)
- Para sair, clique em **"Sair"** no canto superior direito — sua sessão é encerrada

## Perguntas Frequentes

**Perdi minha senha, o que faço?**
Na tela de login, clique em "Esqueci minha senha" (em breve) ou entre em contato para reset manual.

**Posso usar sem internet?**
O app precisa de conexão para carregar e salvar dados. Sem internet, não funciona.

**Quantas notas posso criar?**
No plano atual (Supabase gratuito), o limite é de 500MB de dados — suficiente para centenas de milhares de notas.

**Meus dados são privados?**
Sim. Só você acessa seus dados. Nem mesmo o administrador do sistema consegue ver o conteúdo das suas notas (a menos que acesse diretamente o banco de dados).

**O app funciona no iPhone?**
Sim, funciona em qualquer navegador moderno — Chrome, Safari, Firefox, Edge.

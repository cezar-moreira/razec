📦 Seus dados — onde ficam e como proteger

Como funciona o armazenamento

Navegador (Chrome/Edge) └── localStorage ├── razec_notas → todas as suas notas ├── razec_projetos → seus projetos └── razec_ghToken → token do GitHub ⚠️ Esses dados ficam APENAS neste navegador. Outro navegador = dados diferentes. Outro computador = dados diferentes.
ℹ️ O app funciona 100% offline. Nenhum dado vai para servidores externos. Apenas você tem acesso.

## Estrutura

```
razec/          → App web principal (GitHub Pages)
projetos/       → Subpastas de cada projeto
templates/      → Templates reutilizáveis
.github/        → CI/CD (deploy automático)
```

## Deploy

Push na branch `main` → deploy automático em https://cezar-moreira.github.io/razec
